Introduction

Starlight Charts is a lightweight, dependency-free JavaScript library for creating beautiful, responsive, and interactive charts. It uses the HTML5 Canvas API for rendering, ensuring high performance even with large datasets. The library is designed to be easy to use, with a simple and intuitive API that allows you to create a wide variety of charts with just a few lines of code.

Getting Started

To use Starlight Charts, you need to include the library in your project. You can either download the `starlight-charts.js` file and include it in your HTML, or you can use a package manager like npm or yarn.

Once you have the library included, you can create a chart by creating a new instance of the `StarlightChart` class and passing it a canvas element and a configuration object.

<!-- Include the library -->
<script src="starlight-charts.js"></script>

<!-- Create a canvas element for the chart -->
<canvas id="myChart"></canvas>

<!-- Create the chart -->
<script>
    const ctx = document.getElementById('myChart');
    new StarlightChart(ctx, {
        type: 'bar',
        data: {
            labels: ['Red', 'Blue', 'Yellow', 'Green', 'Purple', 'Orange'],
            datasets: [{
                label: '# of Votes',
                data: [12, 19, 3, 5, 2, 3],
                backgroundColor: [
                    'rgba(255, 99, 132, 0.2)',
                    'rgba(54, 162, 235, 0.2)',
                    'rgba(255, 206, 86, 0.2)',
                    'rgba(75, 192, 192, 0.2)',
                    'rgba(153, 102, 255, 0.2)',
                    'rgba(255, 159, 64, 0.2)'
                ],
                borderColor: [
                    'rgba(255, 99, 132, 1)',
                    'rgba(54, 162, 235, 1)',
                    'rgba(255, 206, 86, 1)',
                    'rgba(75, 192, 192, 1)',
                    'rgba(153, 102, 255, 1)',
                    'rgba(255, 159, 64, 1)'
                ],
                borderWidth: 1
            }]
        },
        options: {
            scales: {
                y: {
                    beginAtZero: true
                }
            }
        }
    });
</script>

Integration with Ruby on Rails

Integrating Starlight Charts into a modern Ruby on Rails application is straightforward, especially with the use of import maps (the default in Rails 7+).

  1. Add the library to your project: Place the `starlight-charts.js` file in your `app/javascript` directory.
  2. Pin the library in `config/importmap.rb`: 
    # config/importmap.rb
pin "application", preload: true
pin "starlight-charts", to: "starlight-charts.js"
  3. Import and use the library in your JavaScript controller: 
    // app/javascript/controllers/chart_controller.js
import { Controller } from "@hotwired/stimulus"
import StarlightChart from "starlight-charts"

export default class extends Controller {
  connect() {
    const ctx = this.element
    new StarlightChart(ctx, {
        // Your chart configuration here
    })
  }
}
  4. Use the controller in your view: 
    <!-- app/views/your_view/index.html.erb -->
<div data-controller="chart">
    <canvas></canvas>
</div>

API Reference

The `StarlightChart` class takes two arguments:

  • context: The canvas element or its 2D rendering context.
  • config: A configuration object that defines the chart's type, data, and options.

The configuration object has the following properties:

  • type: The type of chart to create (e.g., 'bar', 'line', 'pie').
  • data: An object containing the data for the chart.
  • options: An object containing options for customizing the chart.
